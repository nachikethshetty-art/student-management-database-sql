# Student Management Database (SQL)

Project Overview
This project demonstrates the use of SQL and MySQL to build a Student Management Database.
It focuses on database creation, data insertion, and analytical queries to evaluate student performance.

Tools & Technologies
- MySQL
- MySQL Workbench
- SQL

Database Structure
Table: Students
- StudentID (Primary Key)
- Name
- Gender
- Age
- Grade
- MathScore
- ScienceScore
- EnglishScore

SQL Operations Performed
- Create database and table
- Insert student records
- Retrieve student details
- Calculate average scores
- Identify top performers
- Group students by grade and gender
- Filter high-scoring students
- Update records using safe update practices

Key Learning Outcomes
- Practical understanding of SQL queries
- Hands-on experience with MySQL
- Use of aggregation functions
- Safe update handling using primary keys

How to Run
1. Open MySQL Workbench
2. Create a new SQL file
3. Paste the code from `student_management.sql`
4. Execute the script
